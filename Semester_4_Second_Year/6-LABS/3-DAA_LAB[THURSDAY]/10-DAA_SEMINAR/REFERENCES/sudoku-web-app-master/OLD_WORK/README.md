# Sudoku Solver Web App

This Web App Automatically Generates Sudoku Puzzle Based on Various Difficulty Level and Also Informs the Time Taken to Solve Them

### 🍀 Four Levels 

> Easy Level
> Medium Level
> Hard Level
> Random Level
